hexo博客插入图片的方法

发表于 2022-01-23 分类于 学习记录

在写博客的时候遇到了Typora中的图片无法在博客页面中正常显示的问题。百度一下很多回答都是安装hexo-renderer-marked插件。

不过搞了很久还是没有成功。明明是按照步骤一步一步做的竟然搞不好！！

索性换一种解决方案。既然本地的图片无法显示那么我就把图片存储到网络上，这样就可以正常显示图片了。

我的解决方案是Typora + PicGo + Gitee 来管理我需要使用的图片

步骤

1.下载

直接从官网下载就行，相信大家都会。

2.配置

首先需要下载插件，由于PicGo没有对于gitee的配置所以需要自行下载

在PicGo的插件设置一栏输入gitee即可安装。

3.

安装插件之后就可以看到图床设置中出现了gitee的选项

然后需要在gitee创建一个仓库来存放图片。

注意这个仓库一定要是公开的！！！

token需要填私人令牌，在gitee的个人设置里获取

点击生成新令牌就可。

repo就是你那个仓库的URL 比如新建一个images仓库来存放图片，填的就是 UserName/images

4.下面就需要配置一下我们的Typora

点击文件/偏好设置/图像

选择上传图片.. 上传服务选择PicGo 路径就是你安装的位置 然后验证图片上传选项就可以了

经过一番探索终于能在博客中插入图片了，太不容易了！！！

MyFristBlog

发表于 2022-01-23 分类于 学习记录

这是我的第一篇博客，记录一下搭建博客的过程。
踩了许许多多的坑，这一路真的是太不容易了！！

准备工具：

1.git

2.Gitee / GitHub

步骤：

只记录一下大致的步骤，详细的步骤网上很多。

1.先配置git，配置ssh密钥

2.在github新建一个仓库，同时配置ssh密钥

3.安装hexo，node.js，npm等一些工具

4.新建一个空文件夹，在那个文件夹使用hexo init进行初始化

5.hexo clean & hexo g & hexo s （一键三连），实现在本地预览，浏览器输入localhost:4000

如果4000端口被占用，就进行修改端口（详细步骤可以自己搜索）

6.hexo d将博客部署到服务器

通过 UserName.github.io 即可访问

（本来想部署到gitee的，但是身份验证还要手持身份证的照片，多少有点不合适了）
